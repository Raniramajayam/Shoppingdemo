### What Happens Behind the Scenes
```
git add .
  ↓ Moves files to staging area
  ↓ Prepares them for commit

git commit
  ↓ Creates commit object with:
    ├─ Your name & email
    ├─ Current timestamp
    ├─ Your message
    ├─ Snapshot of all files
    └─ Parent pointer (null for first)
  ↓ Stores in .git/objects/

git log
  ↓ Reads .git/objects/
  ↓ Shows commit history
```
